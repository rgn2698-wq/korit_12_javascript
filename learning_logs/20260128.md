## Gemini API ket 발급과정
1. 구글 AI 스튜디오 접속
2. 구글 계정 로그인
3. 왼쪽하단 `Get API Key` 선택
4. 우측상단 `API 키 만들기` 선택
5. 처음의 경우 프로젝트 생성후 만들어야함.
    - 기존에 프로젝트가 있는 경우는 프로젝트 선택으로 API 키를 발급받을수있음.
